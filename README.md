¿Cuál es la diferencia entre autenticación y autorizacion?
La autenticación consiste en comprobar la identidad de una persona, es decir, verificar quién eres.
La autorización, en cambio, se refiere a los permisos o privilegios que una persona tiene una vez autenticada. 

¿Cuál es la función del token JWT en la guía? 
El token JWT funciona como una credencial de acceso que permite al usuario interactuar con los diferentes endpoints del sistema.
